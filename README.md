# Talk to Santa Demo

This demo is using Multimodal Live API feature of Gemini 2.0

# Getting Start using `pip`
create a new environment and follow this command
```
pip install -r requirements.txt
python main.py
```

# Or, getting Start using `uv`
```
uv run main.py
```
